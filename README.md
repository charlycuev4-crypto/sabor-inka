## 📱 ¿Qué es Sabor Inka?

Sabor Inka es un **menú digital** para delivery de auténtica comida peruana en la zona de **Santa Brígida, San Miguel y Trujui** (Buenos Aires, Argentina).

Los clientes pueden:
- Ver el menú completo con fotos y precios
- Armar su pedido con carrito de compras
- Enviar el pedido directo por **WhatsApp**
- Elegir entre **platos especiales** o **menú ejecutivo para fábricas**
